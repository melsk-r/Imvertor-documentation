# 7. Tagged Values

M.b.v. tagged values (TV's) kan aan constructies in UML extra semantiek meegegeven worden. De aanwezigheid van TV's kan m.b.v. metamodellen gefaciliteerd of m.b.v. 
Imvertor afgedwongen worden. In de volgende tabel staan TV's die noch gefaciliteerd noch afgedwongen worden. Daarnaast hebben ze ook geen functie voor |mvertor. Het is 
echter de afspraak dat we onze modellen er van voorzien ter documentatie.
